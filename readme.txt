INSTRUCCIONES DE EJECUCIÓN DE CÓDIGO:

1. El primer paso para ejecutar el código es instalar las librerías que se han usado
en sus correspondientes versiones. Para ello hemos creado requirements.txt, un
fichero con el registro de todas las librerías necesarias en sus respectivas 
versiones. Para instalarlas, basta con ejecutar pip install -r requirements.txt 
en la terminal de Anaconda. Recomendamos al usuario crearse un entorno virtual 
para ejecutar el proyecto, de este modo se evitan posibles cambios de versiones
respecto las que ya existen en local.

2. Para reproducir el proceso, se debe entrar en el fichero test.py y 
ejecutar todas las celdas hasta llegar al apartado de modelización. Es entonces
cuando el usuario puede elegir entre los distintos enfoques de modelización
ejecutando la celda que prefiera.
